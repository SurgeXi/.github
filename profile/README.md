# SurgeXi Business Intelligence

**AI operations platforms that give professionals more capacity — not more headcount.**

SurgeXi builds secure, human-in-the-loop AI systems that take on the repetitive,
high-volume, easy-to-miss work inside a business: intake, classification, follow-up,
routing, document handling, exception detection, and approvals. A **platform core +
vertical packs + connector layer**, with auditability and tenant isolation built in.

*A TeamSmith Enterprises LLC company (Alabama, USA).*

### What we're building
- **Pulse** — the operations platform, delivered as a secured on-node appliance or SaaS.
  Accounting/bookkeeping is the first vertical; the core generalizes.
- **Ma'at Codex** — a verified-knowledge engine for regulated professions: grounded in
  primary authority, verbatim-verified, abstains rather than fabricates.
- **Maestro-Ai / Vertirite** — agent orchestration and AI-governance control planes.
